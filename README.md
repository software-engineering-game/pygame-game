# Pygame Space Dodgers

## Explanation
This repository is for the pygame developed by Team Echo, for Software Engineering Spring 2026 at Kent State University.
The project was coded in Python using the module Pygame.

# Game Name
Space Dodgers

# Controls

## Movement
- Move up: W or up arrow key
- Move down: S or down arrow key
- Move left: A or left arrow key
- Move right: D or right arrow key

## Shooting
- Shoot = Spacebar (hold to shoot continuously)
